SubViewManager

SubViewManager
new SubViewManager(subViewCfg, parent, options)
Local constructor intended for use by Backbone.BaseView. Creates an object specifically for manipulating subviews. Is created automatically by the BaseView instance and accessed through the '.subs' property of the the instance.

 The options or base options you would like to pass to the constructor whenever it is initialized.
location String | HTMLElement <optional>
 If you want to be able to automatically place a subView's DOM element somewhere in the parent view, pass a selector or DOM element or $ instance here that can be used by jquery as a wrapper to append the subView $el
singleton Boolean <optional>
 If you want this view to be a singleton subView. A singleton subView will only allow one instance of it to be created.

underscore_methods()
Partial applications of underscore (or lodash if you use that instead) functions of the same name (operates on the subViews array). List of available underscore methods: 'each', 'map', 'initial', 'rest', 'last', 'first', 'find', 'filter', 'sortBy', 'groupBy', 'where', 'findWhere', 'some', 'every', 'invoke', 'contains', 'max', 'min', 'size', 'without', 'indexOf', 'lastIndexOf', 'isEmpty', 'reject'
Source:
  • backbone-baseview.js
See:
  • http://underscorejs.org
Examples
myBaseView.subs.each(function (subView) {
    console.log('A subView of myBaseView', subView);
});
var view = myBaseView.subs.findWhere({ collection : testCollection });
view.collection === test.collection // returns true
